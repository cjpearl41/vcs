Command Explanation 
compsci@compsci-VBox:~$ sudo apt update

This updates the applications

 
compsci@compsci-VBox:~$ sudo apt install git

This is the line to install git

 
compsci@compsci-VBox:~$ git --version

This is line is used to check the version of git that you have installed

 
compsci@compsci-VBox:~$ git config --global user.name "cjpearl41"

This sets your username for when you pull

 
compsci@compsci-VBox:~$ git config --global user.email "cjpearl41@gmail.com"

This sets your email for when you pull

 
compsci@compsci-VBox:~$ git config --list
                    user.name=cjpearl41
                    user.email=cjpearl41@gmail.com

This shows all the created variables

 
compsci@compsci-VBox:/var/www/html/calebp$ git init

This initializes the .git folder needed for tracking

 
compsci@compsci-VBox:/var/www/html/calebp$ git config --help

This shows you what you can do with git config

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs$ rm -rf .git

This removes the .git folder and version control from a folder

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs$ git add -A

This adds everything from a folder into the staging area

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs$ git status
                    On branch master
                    
                    No commits yet
                    
                    Changes to be committed:
                      (use "git rm --cached <file>..." to unstage)
                    
                        new file:   css/vcs.css
                        new file:   images/workflow.png
                        new file:   index.html

This shows what is ready to be comited and what is needed to be added

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs$ git commit -m "Initial Commit"
                    [master (root-commit) 7a39b9b] Initial Commit
                     3 files changed, 37 insertions(+)
                     create mode 100644 css/vcs.css
                     create mode 100644 images/workflow.png
                     create mode 100644 index.html

This commits the files to be ready to be pushed

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs$ git log
                    commit 7a39b9bd384170010018aa6610f3c94847dff0d9 (HEAD -> master)
                    Author: cjpearl41 <cjpearl41@gmail.com>
                    Date:   Thu May 23 13:32:50 2019 -0600
                    
                        Initial Commit

This shows you past commits

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs$ kate .gitignore

You use a git ignore file to put what things you don't want tracked

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs$ git reset
                    Unstaged changes after reset:
                    M	index.html

This removes added files

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs/GitHubClones/mrm$ mkdir GitHubClones

This makes a directory for clones

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs/GitHubClones/mrm$ git clone https://github.com/CHS-McClendon/VCS_mrm.git mrm

This takes another users repo and clones it into a folder

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs$ git remote add origin https://github.com/cjpearl41/vcs.git

This is were you add to a repo to be able to push to

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs$ git pull origin master --allow-unrelated-histories

This allows you to pull from a repo to get the files

 
compsci@compsci-VBox:/var/www/html/calebp/projects/vcs$ git push origin master

This allows you to push to a repo and update it with the commits
